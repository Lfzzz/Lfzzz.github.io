Twikoo 表情包修改

2022-11-17字数总计:13.1k黔东南州阅读时长:77分钟

因为 Twikoo 本身支持插入表情的功能 所以弄起来没那么复杂

选择表情包生成Json文件链接

可以从 表情速查 https://emotion.xiaokang.me/ 中找到需要的表情系列 选择 Twikoo 可以看到表情的 Json 代码

表情速查截图一
表情速查截图二

保存 Json 文件 并 上传服务器生成访问链接 (下面是我正在使用的表情配置 Json ,你也可以将 img 标签作为 Key , 这样表情包选项卡上就会将默认的文本替换为图片了！)

Twikoo 配置

进入你的 Twikoo 控制台
找到 插件 > SHOW_EMOTION 设置为 true
找到 插件 > EMOTION_CDN 填入 你的Json链接
